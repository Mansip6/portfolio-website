# Mansi Saboo — SRE II Portfolio

A retro pixel-art desktop-style portfolio website for a Site Reliability Engineer II.

## Features

- macOS-inspired desktop UI with menu bar, pixel wallpaper, and dock
- 11 project icons with detailed case studies (beyond resume depth)
- Double-click desktop icons to open project windows
- Dock shortcuts: Home, Work, Skills, Contact, Resume (PDF download)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

Deploy the `dist/` folder to Vercel, Netlify, GitHub Pages, or any static host.

## Tech Stack

- React 19 + TypeScript
- Vite
- Custom pixel-art SVG icons
- Press Start 2P + Inter fonts
